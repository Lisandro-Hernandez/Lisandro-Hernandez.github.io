# 📊 Titration Curves of Strong, Weak and Poly-protic Acids with Strong Base

## 🧠 Overview
Numerically exact titration curves of stron acid (HCl), weak acid (KHP) and poly-protic acid (maleic) with NaOH.

## 🔍 Description
- **Strong-Weak acid Comparison:** Strong acid (HCl) and KHP titrations with NaOH solution.
- **Poly-protic Acid:** Titration cure of maleic acid with NaOH.

## 💡 Key Insights
- These curves are generated by solving the full set of non-linear equilibrium equations, ensuring accuracy even in the extreme dilute limit or near the equivalence points where standard approximations often fail.
